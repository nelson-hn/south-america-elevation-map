# South America Elevation Map (R Project)

This project visualizes the topography of South America using a Digital Elevation Model (DEM) in R.

## Tools used
- R (terra, ggplot2, elevatr, sf)
- SRTM elevation data
- GIS visualization techniques

## Methodology
- Downloaded DEM data for South America
- Processed raster elevation data
- Generated hillshade for terrain realism
- Applied hypsometric color palette
- Produced publication-ready map outputs

## Output
High-resolution elevation maps suitable for academic and professional use.
